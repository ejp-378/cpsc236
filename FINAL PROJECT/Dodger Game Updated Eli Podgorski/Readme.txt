Dodger Game Variation:

NECESSARY FILES:
- 2xpoints.png
- baddie.png
- clearscreen.png
- player.png
- background.mid
- gameover.wav
- dodger.py

CHANGES MADE:
- Shrunk size of player
- Added 2x points powerup
- Added Clearscreen powerup

1. Ensure all necessary files are in the same location as the program to eliminate errors.

2. Select an IDE and run the program

3. Press a key to play

4. Choose a way to move, WASD or mouse (preferably mouse)

5. Avoid 'baddies' for as long as possible

6. Attempt to grab powerups

7. Hit escape to stop playing
