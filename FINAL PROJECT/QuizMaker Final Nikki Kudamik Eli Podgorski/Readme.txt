QuizMaker: 

1. Ensure that the "CPSC 236 TestBank - Sheet1 is in the same folder as the QuizMaker V4 program

2. Open the Quiz Maker V4 program in an IDE of your choice, and run it.

3. Enter your first name

4. Enter your last name

5. Enter your ID, starting with a capital A, followed by 5 integers [1-9]

6. Choose number of questions [10,20]

7. Complete the quiz, selecting a valid answer choice for each question

8. When completed, select 's' to take another quiz or 'q' to exit the program

9. After a quiz is completed, a file will be made in the location of the program with the quiz information.
